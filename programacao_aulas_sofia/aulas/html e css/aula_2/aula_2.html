O que faremos nessa aula?

Na segunda aula de HTML, vamos nos aprofundar na estrutura do HTML introduzindo novos elementos (tags). Apresentaremos as novas tags respondendo as perguntas da nossa primeira atividade de casa.

Questões

  1. O que é um computador?
  2. O que é um software/sistema computacional?
  3. O que é um linguagem de programação?
  4. O que é HTML?
  5. O que é CSS?
  6. O que são tags no HTML?
  7. O que são e para que servem as tags h1, h2, h3, h4, h5 e h6?
  8. O que é a tag img e quais são as principais propriedades?
  9. O que é e para que serve a tag p do HTML?
  10. O que são as tags ul e li? ul e li são siglas para quais termos? Quero que me dê um exemplo de uso das tags ul e li.

Respostas

  1. É um dispositivo eletrônico capaz de processar dados, realizar calculos e executar programas.
  2. É essencial para o funcionamento e a utilidade de um computador, permitindo a execução de diversas funções e aplicações.
  3. É um conjunto de regras que permitem aos programadores escreverem códigos para que o computador execute a tarefa.
  4. É uma linguagem usada para estruturar páginas da web, organizando textos, imagens, links etc, por meio de tags para que os navegadores exibam o conteúdo corretamente.
  5. É uma linguagem usada para definir o estilo e o layout de páginas da web. Trabalha em conjunto com o HTML.
  6. Cada tag do HTML geralmente consiste de uma tag de abertura e uma de fechamento.
    • Tag de abertura: <p>
    • Tag de fechamento: </p>
    • h1: usada para o título principal ou cabeçalho mais importante da página.
    • h2: indica uma seção importante que está logo abaixo do título principal. Pode ser um subtítulo por exemplo.
    • h3: Usada para subtítulos dentro de uma seção h2
    • h4, h5 e h6: representa subtítulos de níveis inferiores, geralmente usados para quebrar o conteúdo em partes menores
  8. É uma tag que serve para exibirmos uma imagem no HTML.
    • src: define o caminho da imagem;
    • alt: fornece uma descrição alternativa para acessibilidade e quando a imagem não é carregada.
    • width e height: ajustam o tamanho da imagem
    • title: exibe uma dica quando o usuário passa o mouse em cima da imagem.
  9. A tag <p></p> no HTML define um parágrafo de texto organizado, separando blocos de conteúdo na página. Ela facilita a leitura e a estrutura do conteúdo.
    1. ul significa unordered list, que é "não ordenada" em inglês
    2. li significa list item, que significa item da lista, que define cada item dentro da lista.
